## ⚡ 3步。30秒。搞定

```
1. Ctrl+Shift+R    → 启用 Remote Control
2. 扫描QR码        → 用手机摄像头
3. 📱 搞定         → 随时随地完全控制
```

无需安装APP。无需创建账户。**扫一扫即用。**

## 📎 发送图片和文件 — 杀手级功能

拍下Bug截图。拍下白板草图。发送设计稿。

**从手机直接附加到AI提示中。**

- 📷 **相机 → AI代理** — 拍照后直接进入对话
- 🖼️ **从相册选择** — 支持同时选择多张图片
- 📄 **任何文件格式** — PDF、CSV、代码文件、文本 — 全部支持
- ⚡ **自动压缩** — 大图通过Canvas压缩，移动网络也能快速上传

无需USB线。无需云上传。无需给自己发邮件。**点一下就发送。**
