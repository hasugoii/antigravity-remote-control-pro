# 🎮 Antigravity Remote Control

> **スマホがあなたのIDE。ストレスゼロ。**

AIエージェントが素晴らしいコードを書いている — でも、あなたは**デスクに縛られて**承認ボタンを待っています。もしソファーに寝転がったまま、コーヒーを飲みながら、**変更を承認、プロンプトを送信、スクリーンショットを添付、ワークフローを実行**できたら？

**それが Remote Control です。**

## 😤 こんな問題ありませんか？

- 🪑 **デスクから離れられない** — AIエージェントがコーディング中、でも毎回承認が必要
- ⏳ **ずっと待機...** — エージェント完了しても、確認するまで気づかない
- 📱 **スマホが役立たず** — 承認もプロンプト送信もできない
- 🤦 **コンテキスト消失** — 5分離席したら、状況がわからなくなる

**Remote Controlがすべて解決します。**

---

## ⚡ 3ステップ。30秒。完了

```
1. Ctrl+Shift+R    → Remote Controlを有効化
2. QRコードをスキャン → スマホカメラで
3. 📱 完了          → どこからでもフルコントロール
```

アプリのインストール不要。アカウント作成不要。**スキャンして即使用。**

| IDE画面を確認 | 📸 スマホで**ライブスクリーンショット**確認 |
| ワークフロー手動再起動 | ⚡ **ワンタップ**でワークフロー実行 |
| エージェントに画像送信不可 | 📎 **写真を撮って**直接送信 |

> 🔥 **結果：** AIエージェントが自律的にコーディング。あなたは**ポケットから監視・操縦するだけ。**真のハンズフリーバイブコーディング。

## 📎 画像・ファイル送信 — キラー機能

バグのスクリーンショットを撮る。ホワイトボードのスケッチを撮影。デザインモックアップを送信。

**スマホからAIプロンプトに直接添付。**

- 📷 **カメラ → AIエージェント** — 写真を撮ったら、そのまま会話に反映
- 🖼️ **ギャラリーから選択** — 複数画像を同時選択
- 📄 **あらゆるファイル形式** — PDF、CSV、コードファイル、テキスト
- ⚡ **自動圧縮** — 大きな画像はCanvasで圧縮、モバイル回線でも高速アップ

USBケーブル不要。クラウドアップ不要。自分宛メール不要。**タップして送信するだけ。**
